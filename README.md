\## Overview

A full-stack web application for managing player registrations for BGMI tournaments.



\## Current Features

\- Player registration form

\- Field validations (BGMI ID, age, required fields)

\- Spring Boot REST APIs

\- Player entity and repository

\- HTML, CSS, JavaScript based UI



\## Upcoming Features

\- Tournament creation and management

\- Admin dashboard

\- Duplicate BGMI ID prevention

\- Better UI/UX improvements

\- Authentication and authorization



\## Tech Stack

\- Java

\- Spring Boot

\- Maven

\- HTML

\- CSS

\- JavaScript

\- Git \& GitHub



\## Status

This project is under development and not production-ready yet.
